# 3DLineDetection
A simple and efficient 3D line detection algorithm for large scale unorganized point cloud. A conference paper based on this code can be found here https://arxiv.org/abs/1901.02532

Prerequisites:
---
1. OpenCV > 2.4.x
2. OpenMP
3. No other libs
